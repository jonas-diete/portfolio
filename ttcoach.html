TIMES TABLE COACH

My game Times Table Coach helps children learn their multiplication tables in a fun and effective way.

Click on the picture to try it!

I built it in 2022 as a final project for Harvard University's online course CS50, using Python, Flask, JavaScript, HTML and CSS.

I learned lots when programming this game. I worked many days on the implementation of the test screen (to test your times table knowledge and earn medals) and learned lots of JavaScript. Many problems had to be overcome, such as programming the countdown timer and storing the progress data.

I made the game fully responsive for desktop, tablet and mobile screens, which required extensive CSS skills.

The game is hosted on Heroku, which required the use of git and correct implementation details so Heroku can run the app successfully.

I am continuously improving the app with what I learned from the Makers course and have recently transitioned to a PostgreSQL database.
