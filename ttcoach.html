TIMES TABLE COACH

My game Times Table Coach helps children learn their multiplication tables in a fun but effective way.

Click on the picture to try it out.

I built it October-December 2021 as a final project for Harvard University's online course CS50, using Flask (a python framework), JavaScript, HTML and CSS.

I learned lots when programming this game. It uses GitHub APIs to save the user data for logins, progress in earning medals and in learning the multiplication facts on GitHub.

I had to write lots of code in python (using the flask framework) and JavaScript (especially for the test and practise screens).

I optimised the game for computer and mobile screens, which required lots of CSS knowledge.

The game is hosted on Heroku, which required the use of git and correct implementation details so Heroku can run the app successfully.
