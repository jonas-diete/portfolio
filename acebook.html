AceBook

During the Makers Course I built this monster-themed 🐲 social media platform as a team project with 5 other developers.

Visit AceBook

Sign up as a new user and post something on our wall!

Screenshots

It uses:

  • Express.js - web framework for Node.js.
  • Nodemon to reload the server automatically.
  • Handlebars to render view templates.
  • Mongoose to model objects in MongoDB.
  • ESLint for linting.
  • Jest for testing.
  • Cypress for end-to-end testing.
