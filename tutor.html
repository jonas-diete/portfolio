In March 2022 I built a website to advertise my qualities as Online German Tutor.

Visit my Tutor Website

I built this website from zero, using my knowledge of HTML, CSS and JavaScript that I gained from CS50x and codecademy's front-end-engineer path.


Features include:

  • The website is fully responsive. I used lots of @media-rules with CSS to achieve a different layout for mobile or tablet screens (the same applies for this portfolio website).
  • It features a timetable of my availability including a timezone calculator, which I programmed using JavaScript.
  • You can send me a message via HTML forms which are then getting processed via flask. I learned how to incorporate Mailgun to securely receive an email (without my email address being displayed to protect against phishing).
  • I learned how to use CSS transitions to animate the menu bar sliding in when the website is viewed on mobile devices (also on this portfolio page).
